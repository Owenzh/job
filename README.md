#Git 基本使用指南

* 第一次clone远程代码

    *git fetch origin master*

* 更新远程项目到本地相当于从服务器拉取代码

    *git pull origin master*

* 检查本地版本状态

    *git status*

* 添加到本地版本控制

    *git add filename*

* 本地提交

    *git commit -m 'comments'*

* 更新本地项目到远程服务器相当于上传本地代码到服务器

    *git push origin master*
